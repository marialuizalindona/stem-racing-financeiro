# Sistema Financeiro para Equipes STEM Racing

## Descrição

Sistema web desenvolvido para auxiliar equipes de STEM Racing no gerenciamento financeiro da temporada.

## Funcionalidades

- Cadastro de patrocinadores
- Controle de receitas
- Controle de despesas
- Metas financeiras
- Dashboard de indicadores
- Relatórios financeiros

## Tecnologias

- Node.js
- Fastify
- PostgreSQL
- Git e GitHub

## Execução

npm install

npm run dev
